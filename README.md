# 🎯 Juego del Ahorcado – Programación Frontend

## 📌 1. Descripción del Proyecto

Este proyecto corresponde a la implementación del juego clásico **“Ahorcado”**, desarrollado utilizando tecnologías de desarrollo frontend.

El sistema selecciona de manera aleatoria palabras relacionadas con **programación frontend**, proporciona una pista y permite al usuario adivinar la palabra antes de agotar sus intentos.

El proyecto fue desarrollado aplicando principios de:

- Manipulación del DOM
- Lógica condicional
- Control de estados
- Diseño responsive (Mobile First)
- Buenas prácticas de organización de código

---

## 🎯 2. Objetivos

- Aplicar JavaScript puro (Vanilla JS).
- Utilizar estructuras de datos (arrays y objetos).
- Implementar eventos y manipulación dinámica del DOM.
- Diseñar una interfaz responsive con Bootstrap.
- Incorporar niveles de dificultad.

---

## 🛠 3. Tecnologías Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript (sin librerías externas)

---

## 📱 4. Enfoque Mobile First

El diseño fue implementado siguiendo el enfoque **Mobile First**, priorizando dispositivos móviles y escalando mediante media queries para tablet y escritorio.

Se utilizan:

- Breakpoints de Bootstrap
- Media queries personalizadas
- Componentes responsivos

---

## 🧪 5. Modos de Juego

El sistema permite seleccionar:

- 🔹 **Modo Normal:** 6 intentos
- 🔹 **Modo Difícil:** 4 intentos

La barra de progreso se ajusta dinámicamente según el modo seleccionado.

---

## 🎮 6. Funcionalidades Implementadas

✔ Selección aleatoria de palabras  
✔ Sistema de pistas  
✔ Control dinámico de intentos  
✔ Representación visual abstracta del estado  
✔ Reinicio del juego  
✔ Modo difícil configurable  
✔ Interfaz responsive  
✔ Código modular y comentado
